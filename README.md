# Dialog Language Model
한국어 Language Model을 활용한 스몰 톡 모델 파인튜닝

## 사용 Language Model
KoBERT, KoGPT-2, KoElectra

## Task
### 1. Text Classifcation
BERT 모델을 이용한 답변 텍스트 클래스 분류 모델.

### 2. Text Generation(Auto Regressive)
GPT-2 모델을 이용한 대화 및 답변 텍스트 생성.
